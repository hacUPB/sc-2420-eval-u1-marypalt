## HARDWARE
Se refiere a todos los componentes físicos de una computadora. Estos componentes trabajan juntos para realizar las funciones de procesamiento de datos y otras tareas.

### COMPONENTES INTERNOS
**CPU (Unidad Central de Procesamiento):** Cerebro de la computadora, encargado de ejecutar instrucciones y procesar datos.

### Memoria

**RAM (Memoria de Acceso Aleatorio):** Almacena temporalmente los datos e instrucciones que usa la CPU.

**ROM (Memoria de Solo Lectura):** Contiene datos permanentes y de arranque.

**Caché:** Memoria de alta velocidad que almacena datos de uso frecuente.


### Dispositivos de Almacenamiento

**Disco Duro (HDD):** Almacenamiento magnético permanente.

**Unidad de Estado Sólido (SSD):** Almacenamiento rápido y sin partes móviles.

**Unidades Ópticas (CD, DVD, Blu-ray):** Almacenamiento basado en discos ópticos.

**Placa Base (Motherboard):** Tarjeta principal donde se conectan todos los componentes del hardware.

**Tarjeta de Video (GPU):** Procesa gráficos y los envía a la pantalla.

**Fuente de Alimentación (PSU):** Proporciona energía a todos los componentes de la computadora.

**Tarjetas de Expansión:** Añaden funcionalidades adicionales a la computadora.

**Sistemas de Refrigeración:** Mantienen los componentes a una temperatura operativa segura.

### COMPONENTES EXTERNOS
### Dispositivos de Entrada

**Teclado:** Permite introducir datos y comandos.

**Ratón:** Dispositivo apuntador para interactuar con la interfaz gráfica.

**Escáner:** Digitaliza documentos y fotos.

**Micrófono:** Captura audio.

### Dispositivos de Salida

**Monitor:** Muestra información visual.

**Impresora:** Produce copias físicas de documentos digitales.

**Altavoces:** Emite sonido.

### Dispositivos de Entrada y Salida (E/S)

**Pantallas táctiles:** Combinan entrada táctil y salida visual.

**Unidades USB:** Almacenamiento externo portátil.

### Dispositivos de Red

**Tarjeta de Red (NIC):** Conecta la computadora a una red.

**Modem:** Convierte señales digitales a analógicas y viceversa 
para la comunicación por teléfono.

**Router:** Distribuye la conexión de red entre múltiples dispositivos.

**Chasis (Case):** Carcasa que alberga y protege todos los componentes internos.
## SOFTWARE
Se refiere a los programas y sistemas operativos que se ejecutan en una computadora y le permiten realizar tareas específicas.
### TIPOS DE SOFTWARE
**Sistema Operativo (OS):** Gestiona el hardware y proporciona servicios básicos para aplicaciones (e.g., Windows, macOS, Linux, Android).

**Software de Aplicación:** Programas diseñados para tareas específicas (e.g., Microsoft Word, Excel, Google Chrome, Adobe Photoshop, Fortnite).

**Software de Desarrollo:** Herramientas para programadores (e.g., Visual Studio, Git, Sublime Text).

**Software de Sistema:** Proporciona servicios básicos y controla el hardware (e.g., controladores, firmware).

**Software de Seguridad:** Protege la computadora contra amenazas (e.g., Norton Antivirus, Malwarebytes).

**Software de Productividad:** Ayuda en tareas eficientes (e.g., Asana, QuickBooks, Evernote).

**Software de Comunicación:** Permite la comunicación entre usuarios (e.g., Zoom, Slack, Facebook).

**Software de Base de Datos:** Gestiona y manipula bases de datos (e.g., MySQL, Oracle Database).

**Software de Virtualización:** Crea entornos virtuales (e.g., VMware, Docker).

## MÁS SOBRE LA CPU  
La CPU (Unidad Central de Procesamiento) es el cerebro de la computadora. Es responsable de ejecutar las instrucciones de los programas, realizar cálculos y gestionar el flujo de datos dentro del sistema informático. La CPU interpreta y ejecuta las instrucciones de los programas almacenados en la memoria principal de la computadora.
### COMPONENTES DE LA CPU
### Unidades
**Unidad de Control (CU):**
Dirige las operaciones de la CPU y coordina cómo se deben ejecutar las instrucciones. Controla el flujo de datos entre la CPU y otros componentes del sistema.

**Unidad Aritmético-Lógica (ALU):**
Realiza operaciones aritméticas (como suma, resta) y lógicas (como AND, OR, NOT). Es el componente donde se llevan a cabo los cálculos matemáticos y las decisiones lógicas.

**Unidad de Decodificación:**
Traduce las instrucciones de los programas en señales que pueden ser entendidas y ejecutadas por otros componentes de la CPU.

**Unidad de Control de Buses:**
Gestiona las conexiones y el flujo de datos entre la CPU y otros componentes del sistema, como la memoria y los dispositivos de entrada/salida.

**Unidad de Predicción de Saltos:**
Intenta predecir el flujo de ejecución de las instrucciones para mejorar la eficiencia y el rendimiento de la CPU, especialmente en operaciones de salto y bifurcación.

**Unidad de Gestión de Memoria (MMU):**
Traduce direcciones de memoria virtual en direcciones de memoria física y gestiona el acceso a la memoria.

### Registros
Pequeñas unidades de almacenamiento dentro de la CPU que contienen datos temporales y de alta velocidad necesarios durante la ejecución de instrucciones. Algunos tipos de registros incluyen:

**Registro de Instrucción (IR):** Contiene la instrucción actualmente en ejecución.
Contador de Programa (PC): Guarda la dirección de la siguiente instrucción a ejecutar.

**Acumulador (ACC):** Almacena resultados intermedios de operaciones aritméticas y lógicas.

**Registros de Propósito General:** Usados para guardar datos temporales y resultados de operaciones.

### Cache
Memoria de alta velocidad ubicada dentro de la CPU que almacena temporalmente datos e instrucciones que se usan con frecuencia, para un acceso rápido. La cache puede ser:

**Cache L1:** Integrada directamente en el núcleo de la CPU, es la más rápida y la más pequeña.

**Cache L2:** Puede estar en el mismo chip que la CPU pero fuera del núcleo, más grande y un poco más lenta que la L1.

**Cache L3:** Compartida entre múltiples núcleos de la CPU, es la más grande y más lenta comparada con L1 y L2, pero aún más rápida que la memoria principal.


## ARQUITECTURA DEL COMPUTADOR
La arquitectura de una computadora es el diseño y la estructura básica de un sistema informático. Define cómo trabajan juntos y se comunican los componentes físicos (hardware) y los programas (software). Esto incluye cómo la CPU, la memoria, y los dispositivos de entrada y salida están organizados y conectados para procesar, almacenar y transmitir datos.

### VARIEDADES DE ARQUITECTURA DEL COMPUTADOR
**Arquitectura de Von Neumann:**
También conocida como arquitectura de programa almacenado. En esta arquitectura, tanto los datos como las instrucciones se almacenan en la misma memoria y se accede a ellos mediante el mismo bus. Esto puede crear un "cuello de botella" debido a la limitada tasa de transferencia entre la CPU y la memoria.

**Arquitectura Harvard:**
Separada en términos de almacenamiento y señales para instrucciones y datos. Permite que la CPU acceda simultáneamente a las instrucciones y a los datos, lo que puede mejorar el rendimiento.

**Arquitectura RISC (Reduced Instruction Set Computer):**
Utiliza un conjunto reducido de instrucciones simples, diseñadas para ejecutarse rápidamente. Ejemplos incluyen los procesadores ARM y algunos procesadores de IBM.

**Arquitectura CISC (Complex Instruction Set Computer):**
Utiliza un conjunto de instrucciones más amplio y complejo, que puede ejecutar operaciones más complicadas en una sola instrucción. Ejemplos incluyen los procesadores x86 de Intel y AMD.
Arquitectura de Computadora con Múltiples 

**Núcleos (Multi-core):**
Integra dos o más unidades de procesamiento (núcleos) en un solo chip. Cada núcleo puede ejecutar tareas de manera independiente, mejorando el rendimiento y la eficiencia energética.

**Arquitectura SIMD (Single Instruction, Multiple Data):**
Utiliza una única instrucción para procesar múltiples datos en paralelo. Es común en procesadores vectoriales y en ciertas aplicaciones gráficas y científicas.

**Arquitectura MIMD (Multiple Instruction, Multiple Data):**
Permite que múltiples instrucciones operen sobre múltiples datos simultáneamente. Es común en sistemas multiprocesador y en clusters de computadoras.
